# Code of Conduct

MPlusForm is an open-source project. Contributors and users are expected to communicate respectfully, stay on topic, and help keep the project useful for the community.

## Expected behavior

- Be respectful and constructive.
- Report bugs with enough detail to reproduce them.
- Discuss technical disagreements calmly.
- Respect user privacy and project safety boundaries.

## Unacceptable behavior

- Harassment or personal attacks.
- Sharing private information without permission.
- Submitting secrets, credentials, or private configuration.
- Encouraging behavior that violates game rules or platform policies.

Maintainers may moderate issues, pull requests, or discussions that do not follow these expectations.
